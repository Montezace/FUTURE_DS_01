# FUTURE_DS_01: Business Sales Performance Analytics

## 🔍 Project Overview
This repository contains my submission for **Task 1** of the Data Science & Analytics Track at Future Interns. The goal of this project is to analyze business sales data to uncover revenue trends, identify top-selling products, and highlight high-performing regional categories.

## 🎯 Objectives
- Clean and organize raw transactional sales data.
- Compute core business KPIs (Total Revenue, Top Categories).
- Visualize revenue trends over time to identify seasonality.
- Generate a client-ready automated HTML report with actionable business insights.

## 🛠️ Tools & Technologies Used
- **Python:** Primary programming language for data manipulation.
- **Pandas:** Data cleaning, aggregation, and time-series grouping.
- **Matplotlib:** Data visualization and trend charting.
- **Google Colab (Jupyter Notebook):** Interactive execution environment.
- **ipywidgets:** For building dynamic, user-friendly data mapping interfaces.

## 📂 Repository Structure
- `untitled_47.ipynb`: The core Google Colab notebook containing the interactive data processing and reporting pipeline.
- `FUTURE_DS_01_Report.html`: The automated, client-ready analytical report generated by the script.
- `data.csv`: The anonymized sales dataset used for this analysis.

## 🚀 How to Run the Project
1. Open the `FUTURE_DS_01_Notebook.ipynb` in Google Colab or a local Jupyter environment.
2. Upload a valid sales dataset (CSV format) when prompted.
3. Use the interactive dropdown menus to map your dataset's columns (Date, Product, Region, Revenue) to the script's logic.
4. Click **Generate Client Report** to automatically process the data and output the final HTML dashboard.
